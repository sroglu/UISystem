**Input**: Feature specification from `/specs/005-uisystem-button/spec.md`

## Summary

Implement M3-style interactive components on top of the foundation layer (WP-1+2+3):

- **WP-4**: `StateLayerController` — plain C# class managing hover/pressed/focused/disabled visual feedback via `SDFRectElement.StateOverlayOpacity` + `RippleElement.StartRipple()`
- **WP-5**: `M3Button` — `[UxmlElement]` VisualElement with 4 variants (Filled/Outlined/Text/Tonal), composed of SDFRectElement + RippleElement + Label + StateLayerController
- **WP-10 partial**: `UISystemMenuItems` — Editor context menu for UXML snippet generation

All colors via `var(--m3-*)` USS tokens. No per-element material overrides. State via C# pointer events (not CSS pseudo-classes).

## Key Technical Decisions

### StateLayerController: Plain C# Class

```csharp
namespace mehmetsrl.UISystem.Core
{
    public class StateLayerController
    {
        private readonly VisualElement _target;
        private readonly SDFRectElement _sdfTarget;  // null if not SDFRectElement
        private readonly RippleElement  _ripple;     // nullable

        private bool _isHovered;
        private bool _isPressed;
        private bool _isFocused;
        private bool _disabled;

        public bool  Disabled    { get; set; }
        public Color OverlayColor { get; set; } = Color.white;

        public StateLayerController(VisualElement target, RippleElement ripple = null);
        public void Attach();
        public void Detach();
    }
}
```

`Attach()` registers `PointerEnterEvent`, `PointerLeaveEvent`, `PointerDownEvent`, `PointerUpEvent`, `FocusInEvent`, `FocusOutEvent`.
`UpdateOverlay()` sets `_sdfTarget.StateOverlayOpacity` to 0.08/0.10/0.0 based on state flags.

### M3Button: [UxmlElement] Composition

```csharp
namespace mehmetsrl.UISystem.Components
{
    [UxmlElement]
    public partial class M3Button : VisualElement
    {
        private SDFRectElement      _root;
        private RippleElement       _ripple;
        private Label               _label;
        private VisualElement       _iconSlot;
        private StateLayerController _stateLayer;

        public event Action OnClick;

        [UxmlAttribute("text")]    public string Text    { get; set; }
        [UxmlAttribute("variant")] public ButtonVariant Variant { get; set; }
        [UxmlAttribute("disabled")] public bool Disabled { get; set; }
    }
}
```

Variant setter applies USS classes: `m3-button--filled`, `m3-button--outlined`, `m3-button--text`, `m3-button--tonal` on `_root`; label color class on `_label`.

### USS Architecture

`button.uss` defines:
- `.m3-button` — base: min-height 40px, padding 10px 24px, align-items center, corner-radius 9999
- `.m3-button--filled` — background `var(--m3-primary)`, shadow from elevation-2 vars
- `.m3-button--outlined` — transparent bg, 1px border `var(--m3-outline)`
- `.m3-button--text` — transparent bg, no border
- `.m3-button--tonal` — background `var(--m3-secondary-container)`
- `.m3-button__label--filled/outlined/text/tonal` — label color per variant

`state-layer.uss` adds: `.m3-disabled { opacity: 0.38; }`
